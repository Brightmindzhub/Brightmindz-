📌 Why Learn HTML?

HTML is the **foundation** of every website. Think of it like this –

  • 🏠 **HTML** is the walls and pillars (structure).
  • 🎨 **CSS** is the paint and decoration (design).
  • ⚙ **JavaScript** is the automation (interactivity).

In short, a website **cannot exist** without HTML! 🚀

🛠 The Best Editors for HTML

You don’t need powerful tools to write HTML! You can easily do it on both **mobile and PC**.

  • 📱 **For Mobile Users:** Pydroid 3 (Best for Android users)
  • 💻 **For PC Users:** VS Code (Recommended), Notepad++ (Simple), Brackets (Web dev-focused)

🛠 Important HTML Elements

HTML has many elements that define a website’s structure. Here are some important ones:

📌 1. Headings (<h1> to <h6>)

                
<h1>This is the Biggest Heading</h1>
<h2>This is a Subheading</h2>

📌 2. Paragraphs (<p>)

                
<p>This is a simple paragraph describing the content.</p>

📌 3. Links (<a>)

                
<a href="https://google.com">Go to Google!</a>

📌 4. Images (<img>)

                
<img src="photo.jpg" alt="My Photo">

📌 5. Lists (<ul> <ol>)

                
<ul>
    <li>Mango</li>
    <li>Banana</li>
</ul>

🚀 Basic HTML Structure

                
<!DOCTYPE html>
<html>
<head>
    <title>My First Website</title>
</head>
<body>
    <h1>Welcome!</h1>
    <p>This is my first webpage.</p>
</body>
</html>

✅ How to Test Your HTML Code

  1. Save the file as index.html.
  2. Double-click it to open in a web browser.
  3. Use **Live Server** in VS Code for instant preview.
