Before writing HTML code, let's understand some basic elements:

  • <html> - The root element of an HTML page.
  • <head> - Contains metadata like the title and description.
  • <title> - Sets the title of the webpage.
  • <body> - The main content area of the page.
  • <h1> to <h6> - Headings (h1 is the largest, h6 is the smallest).
  • <p> - Paragraphs for text.
  • <a> - Links to other pages.
  • <img> - Displays images.
  • <ul>, <ol>, <li> - Lists (unordered and ordered).
